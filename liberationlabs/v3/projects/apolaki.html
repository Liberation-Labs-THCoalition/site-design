Project Apolaki.

Lux prosocialis

Prosocial marketing engine. AI-powered. Ethics-enforced. Hardcoded to be architecturally incapable of producing dark patterns.

View on GitHub → Apply to deploy →

Marketing superpowers, for the Cause

Apolaki gives marketing professionals real capability for the work that matters: web design, social media management, lead generation, media relations, fundraising, brand strategy, and analytics. Ten skill domains, each with built-in ethics guardrails that prevent dark patterns, manipulation, and spam. Named for the Filipino sun god — the work is meant to bring light, not to extract attention.

Ethics, built in not bolted on

False urgency, guilt manipulation, dark patterns, astroturfing, fake reviews, sock puppets, spam, deceptive metrics — all blocked automatically. Financial asks over a threshold, public communications on behalf of an org, press releases, donor contacts, and external data sharing all require human review. When Apolaki blocks something, it suggests the ethical alternative.

Architecture

Request flows through a fast classifier (deny dark patterns / escalate sensitive / allow routine), then to one of the ten skill routers, each scored with the Kintsugi Ethical Framing Engine. A final guardrail check on output. Then either a human review gate or direct delivery.
